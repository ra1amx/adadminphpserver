Instructions    

You can create banners in three ways.

  1. 1. Simple banners: banners made with an image (GIF, JPG or PNG files allowed) and a link. Upload the proper image file (match your banner image size) and fill in the destination link.

  2. 2. HTML5 banners: banners made with HTML, you can use images and javascripts (see more info)

  3. 3. External code: banners made with javascript code that deliver ads from other servers, like Google Adsense code.
